Hello Dart

2023-05-02 | App | Hits

Dart是一種開源的通用程式設計語言。 它最初由Google開發，後來被ECMA批准為標準。

Dart是一種物件導向的語言，具有C語言風格的語法，可以選擇將它編譯成JavaScript。它支援各種程式設計輔助工具。

相關資源

  • DartPad

  1. 變數宣告
    dart主要有四種方式宣告變數，分別為 const、var、type、final

  2. Dart識別符號

    • 識別符號可以包括字元和數位。但是，識別符號不能以數位開頭。
    • 除 _ 或 $ 外，識別符號不能包含特殊符號。
    • 不能是關鍵字。
    • 必須是唯一的。
    • 區分大小寫。
    • 不能包含空格。

  3. 常用型別

    • int, double
    • String
    • bool
    • List
    • Set
    • Map
    • Runes
    • Symbol
    • Null

    基本範例：

    name: BK (String)
    age: 8 (int)
    hobbies: [playing, coding] (JSArray)
